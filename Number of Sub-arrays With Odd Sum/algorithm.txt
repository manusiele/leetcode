 * Key Insight: A subarray sum is the difference between two prefix sums.
 * For subarray [i, j]: sum = prefixSum[j] - prefixSum[i-1]
 * 
 * Odd Sum Property:
 * - odd - even = odd
 * - even - odd = odd
 * - odd - odd = even
 * - even - even = even
 * 
 * Algorithm:
 * 1. Track how many prefix sums are even vs odd
 * 2. For each position, check current prefix sum parity
 * 3. If current is odd: pair it with all previous even prefixes
 *    If current is even: pair it with all previous odd prefixes
 * 
 * Example: arr = [1,3,5]
 * - Start: evenCount=1 (empty prefix), oddCount=0
 * - i=0, num=1: prefixSum=1 (odd)  → add evenCount=1, oddCount=1
 * - i=1, num=3: prefixSum=4 (even) → add oddCount=1, evenCount=2
 * - i=2, num=5: prefixSum=9 (odd)  → add evenCount=2, oddCount=2
 * - Total: 1 + 1 + 2 = 4 ✓
 * 
 * Time Complexity: O(n) - single pass through array
 * Space Complexity: O(1) - only using counters
